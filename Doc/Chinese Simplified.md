# 文档
这是一个 Magisk 模块，用来解决部分 ROM 中中国运营商的 APN 问题。
## 如何使用？
在 Magisk Manager 中安装即可。

**注意：您需要前往 `接入点名称（APN）` 手动 `重置为默认设置`。**
## 它是如何工作的？
用 MIUI 的 `apns-conf.xml` 替换原本系统的。
